### Instalaciones

```shell
# Para crear el entorno virtual
$ python -m venv .venv

# Para ejecutr el entorno virtual
# Linux, Mac --> bash
$ source .venv/bin/activate
# Windows --> PowerShell
$ .venv/Scripts/activate
```
